    Lista com comandos básicos do git

  • git init - Cria um repositório em uma pasta vazia
  • git add - Adiciona modificações do wk (workspace) no stage
  • git commit - Cria um commit no repositório LOCAL
  • git push - Envia as modificações do repositório LOCAL para o remoto
  • git log - Exibe no terminal os commits do repositório no branch atual
  • git branch - Lista as branches do repositório
  • git checkout - Muda de branch
  • git checkout -b < > - Cria e muda para uma nova branch
